**N**o **B**ook Is **W**orthless(세상의 가치없는 책은 없다)의 약자를 따서 NBW라고 작명하였으며 전자상거래의 목적 뿐 아니라 독서 모임 및 여러 독서 관련 강연 이벤트를 자체적으로
